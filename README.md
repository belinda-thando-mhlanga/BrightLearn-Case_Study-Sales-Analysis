# BrightLearn-Case_Study-Sales-Analysis
Sales Case Study - The data is only for one of the products sold at this store.

# The main analysis query
The query will provide comprehensive answers to all case study questions:
1. Daily sales price per unit
2. Average unit sales price
3. Daily % gross profit
4. Daily % gross profit per unit
5. Price elasticity of demand for 3 promotional periods
6. Additional insights and metrics
